# 🎬 YouTube Homepage Clone

A front-end clone of the YouTube homepage built using HTML and CSS. This project focuses on recreating the layout and user interface of YouTube, helping to improve skills in responsive design and modern UI structuring.

## 🚀 Features

* YouTube-style homepage layout
* Header with search bar and icons
* Sidebar navigation menu
* Video grid layout
* Clean and structured UI
* Responsive design (basic)

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```
├── index.html
├── style.css
├── images/
```

## 🎯 Project Goal

The main goal of this project is to practice:

* Layout design using HTML & CSS
* Flexbox / Grid concepts
* UI cloning techniques
* Building real-world front-end projects

Creating UI clones like YouTube helps developers understand how complex layouts are structured and styled in real applications ([Coding Kite][1]).

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/youtube_homepage_clone.git
   ```

2. Open the project folder

3. Run `index.html` in your browser
